# Project-kizuna
Language app
Project KIZUNA: Narrative Japanese Acquisition
Project KIZUNA is a mobile-first, open-source Japanese learning experience designed for ages 10–14. It uses a Narrative-Driven Comprehensible Input (NDCI) model to teach language through a story rather than drills.
🧪 The Science
This app is built on Second Language Acquisition (SLA) principles:
Comprehensible Input (i+1): Content is 90% understandable, with new words introduced through context.
Interaction Hypothesis: Branching dialogue requires learners to interpret meaning, not just translate.
Low Affective Filter: No "failure" states; the game uses natural conversational repairs to keep stress low.
Visual Grounding: Images and icons map sounds directly to concepts, bypassing English translation.
🎮 How to Play
Read & Listen: Follow the dialogue between Kibi (your AI guide) and other characters.
Interact: Choose your response. Your choices shape your personality (Polite vs. Cool).
Learn: Use the "Translation Hints" only when stuck. The goal is to understand the Japanese directly.
